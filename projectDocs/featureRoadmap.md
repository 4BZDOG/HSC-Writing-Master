# HSC AI Evaluator - Feature Roadmap

## 1. Current Capabilities (v2.2.1)

### 🧭 Intelligent Syllabus Navigator
*   **Hierarchical Browsing**: Course > Topic > Sub-Topic > Dot Point > Question.
*   **Discovery Engine**: Automatically detects local course manifests and suggests relevant modules.
*   **Visual Health Rings**: Real-time coverage analytics for syllabus dot points.

### 🧠 AI-Powered Evaluation Engine
*   **Gemini 3 Pro Reasoning**: Upgraded marking logic using 128k context and thinking budgets.
*   **Ruthless Marker Persona**: Strict adherence to NESA verbs; no "participation marks" for off-verb responses.
*   **Dynamic Rubric Synthesis**: Generates descending 1-N mark rubrics on demand for any question.
*   **The Improvement Loop**: Guided "N+1" band upgrades with side-by-side comparison.

### ✍️ Writing Workspace
*   **Chromatic Editor**: Visual theme shifts from Slate to Indigo based on response quality.
*   **Live Metrics**: Dashboard tracking word count, keywords, and structural signposts.
*   **Reference Pane**: Instant access to Band Descriptors and tailored Marking Criteria.

### 🛠️ Admin & Data Tools
*   **Content Audit Studio**: Bulk-generate questions, rubrics, and samples for entire topics.
*   **Data Vault**: Integrated Time Machine for backups and full JSON Import/Export with conflict resolution.
*   **Circuit Breaker**: API Guard monitoring error rates (429s) to prevent API lockout.

## 2. Future Planned Enhancements

### Phase 1: Longitudinal Analytics (In Progress)
*   **Student Progress Radar**: Visualise improvement across different cognitive tiers over time.
*   **Weakness Heatmap**: Automated detection of "difficult" verbs or syllabus modules based on historical scores.

### Phase 2: Multimodal & Interactive
*   **Multimodal OCR**: Upload photos of handwritten exam papers for AI transcription and marking.
*   **Socratic Mode**: A toggle for the evaluator to ask "guiding questions" instead of giving direct answers.
*   **Audio Synthesis**: Natural voice reading of "Summative Commentary" for auditory learners.

### Phase 3: Collaborative Ecosystem
*   **Global Library**: Optional opt-in to a community-shared repository of high-quality questions.
*   **Classroom Sync**: Secure sharing of student progress dashboards with teachers.

### Phase 4: Subject Expansion
*   **Domain-Specific Rubrics**: Custom reasoning modules for English (PEEL), History (Historiography), and Mathematics (Working).
